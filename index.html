html&css Cheat Sheet

HTML

Comentários

Sintaxe: <--!Comentário Exemplo -->

Headers

Sintaxe:<h1>Header Exemplo</h1>

Headers vão de h1 até h6

Paragráfos

Sintaxe: <p>Parágrafo Exemplo</p>

Spans

Sintaxe: <span>Span Exemplo</span>

Spans são utilizados para conteúdos inline

Anchor

Sintaxe: <a>Exemplo mensagem link</a>

Utilizados dentro da tag:

  • href="Exemplo de link externo"
  • href="#Exempo de link interno"
  • href="#" Para dead links
  • target="_blank" Para abrir o link em uma nova aba

Imagens

Sintaxe: <img>

Pode ser utilizada entre o <a> para ser usada como link

Utilizados dentro da tag:

  • src="Exemplo de link da imagem"
  • alt="Exemplo descrição da imagem"

Listas

Sintaxe: <ul><li>Para lista sem ordem</li></ul> ou <ol><li>Para lista com ordem</li></ol>

Uma tag <li></li> para cada item da lista

Input

Sintaxe: <input>

Utilizado para pegar um input do usuário

Utilizados dentro da tag:

  • type="text" Para inputs de texto
  • placeholder="Exemplo de placeholder" Para mensagem para padrão que vai estar no campo de texto
  • type="radio" Para radio buttons (só pode ser selecionada uma opção
  • type="checkbox" Para checkbox (pode ser selecionada mais de uma opção
  • name="Exempo de nome" Boa prática e vincular os radio buttons e checkbox, deve ser utilizado o mesmo nome para todos itens do grupo
  • value="Exemplo de valor para resposta" Valor que vai aparecer quando a resposta selecionada for enviada
  • checked Para item padrão que vai estar selecionado
  • required Para campo obrigatório

Label

Sintaxe:<label><input type="radio" ou type="checkbox"></label>

Utilizado para mostrar o nome da opção em radio buttons e checkbox

Utilizados dentro da tag

  • for="id do radio button ou checkbox relacionado" Para boas práticas

Form

Sintaxe:<form><input></form>

Para envio de formulários

Utilizados dentro da tag:

  • action="Exemplo de link" Link para enviar as respostas do formulário

Button

Sintaxe:<button>Nome do botão</button>

Utilizados dentro da tag

  • type="submit" Para envio de formulários

Doctype

Sintaxe:<!DOCTYPE html>

Sintaxe:<html>Todo conteúdo do html</html>

Sintaxe:<head>Meta dados: Link, meta, title e style</head>

Sintaxe:<body>Conteúdo da página</body>

Doctype sempre no ínico do documento, abertura do html logo em seguida. Para quem usa vscode pode ser utilizado o comando html:5 para formato de documento padrão
