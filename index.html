html&css Cheat Sheet

HTML

Comentários

Sintaxe: <--!Comentário Exemplo -->

Headers

Sintaxe:<h1>Header Exemplo</h1>

Headers vão de h1 até h6

Paragráfos

Sintaxe: <p>Parágrafo Exemplo</p>

Spans

Sintaxe: <span>Span Exemplo</span>

Spans são utilizados para conteúdos inline

Formatação de texto

  • <strong>Bold</strong>
  • <u>Uderline</u>
  • <em>Italic</em>
  • <s>Taxado</s>
  • <hr> Para traçar uma linha da largura do container

Anchor

Sintaxe: <a>Exemplo mensagem link</a>

Utilizados dentro da tag:

  • href="Exemplo de link externo"
  • href="#Exempo de link interno"
  • href="#" Para dead links
  • target="_blank" Para abrir o link em uma nova aba

Imagens

Sintaxe: <img>

Pode ser utilizada entre o <a> para ser usada como link

Utilizados dentro da tag:

  • src="Exemplo de link da imagem"
  • alt="Exemplo descrição da imagem"

Listas

Sintaxe: <ul><li>Para lista sem ordem</li></ul> ou <ol><li>Para lista com ordem</li></ol>

Uma tag <li></li> para cada item da lista

Input

Sintaxe: <input>

Utilizado para pegar um input do usuário

Utilizados dentro da tag:

  • type="text" Para inputs de texto
  • placeholder="Exemplo de placeholder" Para mensagem para padrão que vai estar no campo de texto
  • type="radio" Para radio buttons (só pode ser selecionada uma opção
  • type="checkbox" Para checkbox (pode ser selecionada mais de uma opção
  • name="Exempo de nome" Boa prática e vincular os radio buttons e checkbox, deve ser utilizado o mesmo nome para todos itens do grupo
  • value="Exemplo de valor para resposta" Valor que vai aparecer quando a resposta selecionada for enviada
  • checked Para item padrão que vai estar selecionado
  • required Para campo obrigatório

Label

Sintaxe:<label><input type="radio" ou type="checkbox"></label>

Utilizado para mostrar o nome da opção em radio buttons e checkbox

Utilizados dentro da tag

  • for="id do radio button ou checkbox relacionado" Para boas práticas

Form

Sintaxe:<form><input></form>

Para envio de formulários

Utilizados dentro da tag:

  • action="Exemplo de link" Link para enviar as respostas do formulário

Button

Sintaxe:<button>Nome do botão</button>

Utilizados dentro da tag

  • type="submit" Para envio de formulários

Doctype

Sintaxe:<!DOCTYPE html>

Sintaxe:<html>Todo conteúdo do html</html>

Sintaxe:<head>Meta dados: Link, meta, title e style</head>

Sintaxe:<body>Conteúdo da página</body>

Doctype sempre no ínico do documento, abertura do html logo em seguida. Para quem usa vscode pode ser utilizado o comando html:5 para formato de documento padrão

CSS

CSS Selectors

  • .class Para selecionar pela classe
  • #id Para selecionar pelo id
  • elemento Para selecionar pelo elemento
  • !important após declaração para dar prioridade, exemplo: color: red !important

Formatações Básicas

  • color: cor Para mudar cor do texto
  • font-size: Para mudar o tamanho da fonte
  • font-family: Para mudar a família da fonte
  • background-color: Para mudar a cor de fundo

Resize

  • widht: largura
  • height: altura
  • length: comprimento

Box Model

...-top, ...-right, ...-bottom, ...-left para mudar lados individualmente

clockwise notation: padding: top right bottom left

Border

  • border-color: cor da borda
  • border-width: largura da borda
  • border-style: solid
  • border-radius: 50% for a circle

Padding

Margin

Margens negativas fazem caixa aumentar de tamanho

Absolute and Relative units

Absolute

  • px - pixels
  • in - inches / polegadas
  • mm - milimetros

Relative

  • rem / em - based on the font size

Colors

  • HEX: color:#00AAFF - abreviado: #0AF
  • RGB: color:rgb(255, 255, 255)

Variable

Declarar no :root para escopo global, ou na seleção desejada, cor da variável: --nome-variável: blue;

Utilizar color: var(--nome-variável, cor reserva);

Váriaveis podem ser alteradas dentro de media querys

Formatação de texto

  • text-align: justify;
  • text-align: center;
  • text-align: right;
  • text-align: left;

Propriedades

  • box-shadow: offset-x offset-y blur-radius spread-radius color
  • opacity: 0 to 1
  • text-transform: lowercase uppercase capitalize initial inherit none
  • line-height: Entrelinha
  • elemento:hover para editar o elemento on hover

Positions

position: static;

Posição padrão dos elementos e não é influenciada por top, right, bottom e left

position: relative;

Muda conforme a posição inicial do elemento, para top, right, bottom e left

position: fixed;

Muda conforme viewport, ou seja não é afetado pelo scroll e valores afetam com base nas bordas do viewport

position: absolute;

Muda conforme o parent container e pode sobrepor outros elementos

float: left ou right;

Vai para o canto do parent container

z-index: 1

Joga o elemento para frente ou para trás quando for sobreposto
